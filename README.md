# Project 1

### ENGO 651 - Adv. Topics on Geospatial Technologies


For this Lab 1 project I built a flask web application to search book and find details about books from postgres database.

##### Python Requirments: 
 Markup : 1. Flask
          2. psycopg2
          3. werkzeug

##### Tables:

Users | Books | Reviews
| :--- | ---: | :---:
id  | isbn | id
username  | title | username
email  | author | review
password  | year | isbn
